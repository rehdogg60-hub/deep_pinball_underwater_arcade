# 🐙 REH DOGG: DEEP PINBALL – Underwater Arcade

A neon-drenched, physics-driven pinball game set in the abyss. Flip, bounce, and combo your way through glowing sea creatures, kraken targets, and escalating multipliers.

---

## 🌊 GAME FEATURES

- **Fully simulated pinball physics** – gravity, flipper impulse, bumper knockback
- **Underwater theme** – coral flippers, sonar grid, bubble particles, deep sea gradient
- **Dynamic scoring system** – base points × multiplier (up to 6x)
- **Kraken bonus target** – resets multiplier + adds big points
- **Mobile & desktop support** – keyboard (A/S, Space) + on‑screen touch buttons
- **3 balls per game** – classic pinball rule set

---

## 🎮 HOW TO PLAY

| Action            | Keyboard         | Touch / Mobile              |
|-------------------|------------------|-----------------------------|
| Left flipper      | `A` key          | Left on‑screen button       |
| Right flipper     | `S` key          | Right on‑screen button      |
| Launch ball       | `Space`          | “LAUNCH BALL” button        |
| New game          | (on‑screen)      | “NEW GAME” button           |

### Objective
- **Keep the ball in play** – if it falls past the bottom flippers, you lose a ball.
- **Hit glowing bumpers** (🐡, 🐙, 🪸, 🐠, 🦀, 🧜‍♀️) to earn points.
- **Smash static targets** (💎, ⚓, 👑 KRAKEN) for huge bonus points.
- **Build your multiplier** – kraken targets and mermaid bumpers can increase your multiplier up to **6×**.
- **Survive with 3 balls** – highest score wins.

---

## 🔧 INSTALLATION & RUN

1. Save the HTML file as `deep-pinball.html`
2. Open with any modern web browser (Chrome, Firefox, Safari, Edge)
3. No internet connection needed – works entirely offline

> **Note:** Audio uses Web Audio API and requires a user gesture (first click/tap) to enable sound on some browsers.

---

## 🕹️ ARCADE TIPS

- **Use both flippers** – timing is everything to redirect the ball toward bumpers.
- **Aim for the kraken target** (top center 👑) – it resets multiplier to a higher level and awards 1000+ points.
- **The mermaid bumper** (🧜‍♀️) randomly increases your multiplier – hit it often.
- **Don't launch too hard** – a soft launch can sometimes trap the ball in bumper clusters for repeated hits.

---

## 🎨 VISUAL & SOUND DESIGN

- **Glow effects** – neon blue / cyan shadows on all active elements
- **Particle system** – splashes and bubbles on bumper hits
- **Retro arcade HUD** – score, multiplier, balls left
- **Original sound FX** – flips, bumper hits, kraken bonus, game over

---

## 🐛 KNOWN ISSUES / LIMITATIONS

- Ball may rarely clip through flippers at very high speeds – physics step is locked at 60fps to minimise this.
- Touch buttons are large enough for most screens; on very small devices, use landscape orientation.
